# GT PLANET AUTO PARTS SDN BHD Website

This is the official company website for **GT PLANET AUTO PARTS SDN BHD**.

The website is built with **Next.js** and is designed to present the company’s information, services, auto parts, gallery, and contact details.

## Project Overview

This project serves as the official website for GT PLANET AUTO PARTS SDN BHD, an auto parts company.

The website allows visitors to learn more about the company, explore its services, view auto parts-related information, browse the gallery, and contact the company.

## Pages

The website includes the following main pages:

* Home
* About
* Services
* Parts
* Gallery
* Contact

## Tech Stack

* Next.js
* Node.js
* GitHub
* Vercel

## Getting Started

To run this project locally, follow the steps below.

### 1. Clone the repository

```bash
git clone https://github.com/ETHAN071104/GTPlanet.git
```

### 2. Navigate to the project folder

```bash
cd <your-project-folder>
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the development server

```bash
npm run dev
```

### 5. Open the project in your browser

```bash
http://localhost:3000
```

## Deployment

This project is deployed using **Vercel**.

The source code is managed on **GitHub** and can be connected to Vercel for deployment.
